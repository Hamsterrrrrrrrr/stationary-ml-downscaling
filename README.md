# stationary-ml-downscaling
**Mitigating Non-stationarity in Machine Learning–Based Downscaling of Climate Projections**

This repository contains code for data preprocessing, model training, and analysis/figure generation used in the paper:

> *Mitigating Non-stationarity in Machine Learning–Based Downscaling of Climate Projections*

The goal is to improve robustness of ML-based statistical downscaling under non-stationary climate conditions (e.g., distribution shift between historical and future periods).
